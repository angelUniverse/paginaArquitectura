Unidad 1: Estructura, organización y arquitectura de computadoras

1.1 Definición de los conceptos de arquitectura, organización y estructura

La arquitectura de computadoras se refiere a aquellos atributos del sistema visibles para el programador o que tienen un impacto directo sobre la ejecución lógica de un programa. Esto incluye el conjunto de instrucciones, el número de bits usados para representar diversos tipos de datos, los mecanismos de entrada/salida y las técnicas de direccionamiento.

La organización de computadoras se refiere a las unidades operativas y sus interconexiones que realizan las especificaciones de arquitectura. Por ejemplo, señales de control, interfaces entre computadora y periféricos, y la tecnología de memoria utilizada.

La estructura se refiere a la forma en que se relacionan entre sí los componentes individuales de un sistema de computación.

1.2 Función y estructura de un computador

Una computadora puede verse como una máquina de procesamiento de datos. La función general de una computadora es procesar datos de manera que sea útil. Se puede dividir en cuatro funciones principales:

  • Procesamiento de datos
  • Almacenamiento de datos
  • Movimiento de datos
  • Control

1.3 Interacción entre los componentes de un sistema de cómputo

Los principales componentes de un sistema computacional son:

  • Unidad Central de Procesamiento (CPU)
  • Memoria principal
  • Dispositivos de entrada/salida (E/S)

Estos componentes interactúan a través del sistema de buses, que incluye el bus de datos, el bus de direcciones y el bus de control.

1.4 Estructura de topologías de buses

El bus es un medio de comunicación entre los componentes de una computadora. Existen varias topologías, entre las más comunes:

  • Topología de bus simple: todos los módulos comparten un mismo conjunto de líneas de comunicación.
  • Topología en estrella: cada módulo se conecta a un controlador central.
  • Topología en anillo: cada módulo tiene una conexión directa con el siguiente formando un anillo.

1.5 Tipos de buses

Los tipos principales de buses en un sistema computacional son:

  • Bus de datos: transporta la información entre los componentes.
  • Bus de direcciones: indica la dirección de la fuente o el destino de los datos.
  • Bus de control: transporta señales de control y sincronización.
